📲 Instalar como App
✕
Hapirium funciona sin internet una vez instalada
Los datos se guardan localmente en tu dispositivo de forma segura.
iPhone / iPad (Safari)
1. Abre esta página en Safari
2. Toca el ícono de compartir ⎙ (parte inferior)
3. Selecciona "Añadir a pantalla de inicio"
4. Toca "Añadir" en la esquina superior derecha
Android (Chrome)
1. Abre esta página en Chrome
2. Toca el menú ⋮ (esquina superior derecha)
3. Selecciona "Instalar app" o "Añadir a inicio"
4. Confirma tocando "Instalar"
Computador (Chrome/Edge)
1. Busca el ícono de instalación ⊕ en la barra de URL
2. Haz clic en "Instalar"

📥 Datos y Exportación
✕
💾 Backup Completo
Guarda TODOS tus datos en un archivo JSON. Úsalo para hacer copias de seguridad o moverlos a otro dispositivo.
📤 Restaurar / Importar
Carga un backup de Hapirium para restaurar datos o sincronizar con otro dispositivo. Reemplaza los datos actuales.
🔄 Cómo sincronizar entre dispositivos
1. En este celular: toca Descargar Backup
2. Comparte el archivo .json por WhatsApp, Drive o email
3. En el otro dispositivo: abre Hapirium → Restaurar → selecciona el archivo
4. ¡Listo! Todos los datos quedan sincronizados
